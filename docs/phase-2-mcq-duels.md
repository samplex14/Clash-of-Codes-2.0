# Phase 2 — MCQ Duels (Matchmaking Rounds)

## Overview

The main tournament bracket phase. The 64 qualifiers from Phase 1 battle head-to-head in a **single-elimination** MCQ knockout.

## Format

- Top 64 qualifiers are **randomly paired** into 1v1 duels.
- Both players in a duel receive the **exact same set of questions simultaneously** on their screens.
- The player who correctly answers more questions **wins** and advances.

## Tiebreaker

If both players answer the same number of questions correctly, the player who **completed their answers in less time** is declared the winner.

## Elimination

- The **losing player is immediately eliminated** — no second chances, no consolation rounds.
- Each round halves the remaining field.

## Round Progression

| Round          | Players In | Players Out | Players Advancing              |
| -------------- | ---------- | ----------- | ------------------------------ |
| Round of 64    | 64         | 32          | 32                             |
| Round of 32    | 32         | 16          | 16                             |
| Round of 16    | 16         | 8           | 8                              |
| Quarter-Finals | 8          | 4           | 4                              |
| Semi-Finals    | 4          | 2           | 2                              |
| Last Round     | 2          | 1           | 1... wait — top 8 go to finals |

> Phase 2 ends when exactly **8 players** remain.

## Key Points

| Detail                | Value                   |
| --------------------- | ----------------------- |
| Starting participants | 64                      |
| Match style           | 1v1 MCQ duel            |
| Elimination           | Single elimination      |
| Tiebreaker            | Fastest completion time |
| Finalists             | Top 8                   |

## Next Phase

The surviving 8 players advance to [Phase 3 — Grand Finals](./phase-3-grand-finals.md).
